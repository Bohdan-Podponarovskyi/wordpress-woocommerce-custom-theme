Module: SwiperUtils

Utilities for initializing and managing Swiper instances

Requires

  • module:swiper

Methods

(static) exports.destroySwiper(instance)

Destroy one or multiple Swiper instances. Maybe will be used in the future

Parameters:
Name Type Description
instance Swiper | Array.<Swiper>

Swiper instance(s) to destroy
Example
// Destroy a single swiper instance
destroySwiper(mySwiper);

// Destroy multiple swiper instances
destroySwiper(allSwipers);

(static) exports.initMultipleSwipers(selector, optionsopt) → {Array.<Swiper>}

Initializes Swiper instances for multiple elements

Parameters:
Name Type Attributes Default Description
selector string

CSS selector for multiple Swiper containers
options Object <optional>
 {}

Swiper configuration options
Returns:

Array of Swiper instances

Type
Array.<Swiper>
Example
// Initialize all swipers with the same options
const allSwipers = initMultipleSwipers('[data-swiper-container]', {
  loop: true,
  autoplay: {
    delay: 3000
  }
});

(static) exports.initSwiper(selector, optionsopt) → {Swiper}

Initializes a new Swiper instance with navigation module

Parameters:
Name Type Attributes Default Description

data-attribute for Swiper container
options Object <optional>
 {}

Optional Swiper configuration options
Returns:

New Swiper instance

Type
Swiper
Example
// Initialize with custom options
const customSwiper = initSwiper('[data-swiper-container]', {
  slidesPerView: 3,
  spaceBetween: 20
});
