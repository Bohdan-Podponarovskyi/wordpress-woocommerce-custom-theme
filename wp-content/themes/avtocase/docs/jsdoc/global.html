(constant) onDomReady

Executes a callback function when the DOM is fully loaded and ready. If DOM is already loaded, executes the callback immediately. Otherwise, waits for the DOMContentLoaded event.

Example
onDomReady(() => {
    // DOM is ready, you can safely access the DOM
    myFunction();
    // or
    const element = document.querySelector('.my-element');
});
